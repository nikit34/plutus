# Security policy

## Reporting a vulnerability

Please report security issues **privately** so they can be fixed before public disclosure:

- **Preferred:** open a private advisory at <https://github.com/nikit34/plutus/security/advisories/new>
- **Email:** `noreply@plutus.firstmessage.ru`

Include reproduction steps, affected version/commit, and the impact you observed. You will get an acknowledgement within 72 hours.

## Scope

The hosted reference deployment at <https://plutus.firstmessage.ru> and the source code in this repository.

Out of scope: third-party providers (Stripe, NOWPayments, Resend), self-hosted forks of this code, or vulnerabilities reachable only with admin/seller credentials you control.

## Disclosure

We aim to release a fix within 14 days for critical findings, 30 days for high, and 90 days for everything else. Coordinated disclosure is the default — please don't publish details until a fix is available.
